# FLM Hukuk Dil Kartları

Bu sürüm yeni `Flm_En_Tr_Lng.xlsx` dosyasını ana kelime havuzu olarak kullanır.

## İçerik
- Kelime sayısı: 7500
- Kategori sayısı: 15
- Kanun/metin kaynak sayısı: 9
- Çevrimdışı PWA çalışma
- LocalStorage ile kayıt olmadan kişisel ilerleme
- Firebase config eklenirse Google hesabı ile senkron mimarisi

## Kurulum
1. Klasörü telefona kopyalayın veya basit bir HTTP sunucuda açın.
2. `index.html` açılır.
3. Firebase istenirse `firebase-config.js` doldurulur.

## Not
Cümle/paragraf çevirileri için ayrı EN/TR çeviri veri havuzu verilmediği için uygulama otomatik çeviri yapmaz. Alanlar önceden depolanabilir ve düzenlenebilir yapıdadır.
